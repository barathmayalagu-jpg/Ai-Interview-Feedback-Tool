# AI Interview Feedback Tool

A simple front-end prototype for an AI-powered interview feedback experience.

## Pages
- Home page
- Login and registration
- Dashboard
- Interview session
- Feedback view

## Structure
- HTML pages in the project root
- Shared styles in css/
- Interactive scripts in js/
- Sample interview questions in data/questions.json

## Run locally
Open index.html in your browser to view the landing page.
